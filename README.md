# E-Commerce Store (PHP + MySQL)

This is a fully functional e-commerce website built as a university assignment. It allows users to browse products, register an account, add items to a cart, and place orders. Admins can manage products and view orders. This is a small project built for education purposes only.

## Features
- Product catalog
- Shopping cart
- User registration & login
- Order placement
- Admin dashboard 

## Programming Languages and Database
- PHP
- MySQL
- HTML, CSS, JavaScript
